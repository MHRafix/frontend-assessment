## Getting Started

- Frontend assessment of Property Management Dashboard

- Technology I Used:-

* Next.JS
* TypeScript
* TailwindCSS
* MantineUI
* React-hook-form
* Yup validation
* LocalStorage
* ApexChart
* Mantine-React-Table
